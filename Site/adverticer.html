Advertice on AidsAD

AidsAD is operated in it's Github Repository if you want to Advertice on AidsAD you have to sign into github
We're not using a dashboard system on AidsAD

How to submit your AD

To submit your ad you have to open a pull-request that contains all the ad material you want to submit. after you submited we will check if it follows all our rules

What you need to know:

Web-Display (Image)
Acceptable Formats: png, jpg, gif
Animations allowed
Include redirect link, catagory name and resolution
Maximum File-Size: 1,5MB


Web-Display (Iframe)
Acceptable Formats: html, css
External Libraries are prohibited
Include redirect link, catagory name and resolution
Maximum File-Size: 1,5MB


Text-Display:
Acceptable Formats: strings
Include link and catagory name
Maximum String-Size: 40-100 characters
